1) instalar as bibliotecas necessárias:

	pip install playsound==1.2.2
	pip install pygame

2) executar o ficheiro efolio_B.py

3a) (menu principal) Para jogar: carregar em "Start the game!"
3b) (segundo menu) carregar em "1vs. 1"	

4) As primeiras peças a jogar são as vermelhas - use o rato e selecione uma peça e depois o local para onde ir

5) Agora, com as peças pretas, faça o mesmo

6) Quando o jogo acabar, quer na consola, quer no ecrã surge uma mensagem - o jogo volta ao menu principal sozinho

7) Verifique o histórico das partidas na botão "Stats"



Em caso de dúvidas: veja o vídeo de demonstração nesta pasta (Demonstração de utilização.mp4)
